Global Attributes

class: Specifies one or more class names for an element.
id: Specifies a unique id for an element.
style: Used to apply inline CSS styles.
title: Provides additional information about the element (often displayed as a tooltip).

Attributes for form control

name: Defines the name of the form control, used for form submission.
value: Specifies the initial value of the form control.
placeholder: Provides a short hint that describes the expected value of the form control.
required: Indicates that the form control must be filled out before submitting the form.
disabled: Disables the form control, making it non-interactive.
readonly: Specifies that the form control is read-only (cannot be modified).
checked: Indicates whether a checkbox or radio button should be checked by default.

Table Attributes

colspan and rowspan: Specify the number of columns or rows a table cell should span.
border: Sets the thickness of the border around a table.
align: Specifies the horizontal alignment of a table or table cell.

Multimedia Attributes

controls: Adds playback controls to the media element.
autoplay: Automatically starts playback of the media.
loop: Loops the media playback.
preload: Specifies how the media file should be preloaded.
muted: Mutes the audio by default.
volume: Sets the default volume level (0.0 to 1.0).
controlsList: Specifies which controls should be displayed in the audio player.
Values: "nodownload", "nofullscreen".
