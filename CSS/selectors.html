CSS Selectors

element selector

Syntax :
element(tag) {

}

#id selector

Syntax :
#id_value {

}

.class selector

Syntax :
.class_value {

}

* selector (univesal selector)

note: it applies to all the tags individually.

Syntax :
* {

}

attribute selector

Syntax :
[attribute_name] {

}
example: src, href, alt, class, id

Pseudo-class

Syntax :
selector:pseudo-class {

}
example: hover, active, visited, first-child, last-child, nth-child
